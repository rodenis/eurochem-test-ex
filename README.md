### EuroChem test task

#### How to setup:

1) install dependencies: `npm install`

2) build client code bundle: `npm run build`

3) create orm config in `config/orm.json` (example file is `config/orm.example.json`)

4) start server: `npm start`

#### Run tests:

```
npm run test
```

#### Run lint:

```
npm run lint
```
